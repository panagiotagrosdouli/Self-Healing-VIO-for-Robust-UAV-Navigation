# Abstract

Visual-Inertial Odometry is a core capability for GPS-denied autonomy, but downstream navigation may become unsafe when the estimator silently degrades. SHIELD-VIO studies uncertainty-aware monitoring and failure shielding for VIO. The current repository provides a research software scaffold for error-state propagation, visual quality monitoring, IMU consistency checks, uncertainty metrics, risk normalization, and conservative fallback decisions. It does not claim benchmark superiority; real-data evaluation is pending.
