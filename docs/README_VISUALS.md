# README visuals

The main README embeds the generated SVG overview at:

`assets/readme/shield_vio_research_overview.svg`

The graphic is maintained as repository-native SVG so it renders without external hosting, remains diffable, and can be updated alongside implementation status changes.

The visual summarizes:

- camera and IMU inputs;
- real sparse feature tracking;
- bias-aware IMU preintegration;
- the 15-state error-state EKF;
- consistency and health diagnostics;
- calibrated and conformal failure prediction;
- domain-shift awareness;
- stateful navigation shielding;
- protective and recovery actions;
- current dataset and hardware evidence boundaries.

The graphic is explanatory documentation. It does not constitute experimental evidence or imply that pending dataset and hardware validation has been completed.
