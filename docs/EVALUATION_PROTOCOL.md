# Evaluation Protocol

Metrics: ATE, RPE, NEES, NIS, failure detection precision, recall, false alarm rate, time-to-detection, tracking failure rate, shield activation rate, and runtime FPS.

Baselines: standard VIO without shield, fixed uncertainty threshold, visual quality monitoring, SHIELD-VIO uncertainty-aware monitor, and oracle degradation labels.

Reporting rule: do not report benchmark numbers unless the config, seed, dataset sequence, commit hash, and result files are available.
